# Coberturas Electricidad

Guía interactiva de coberturas de electricidad para reparadores colaboradores.

## 📱 Instalar como app

### Android (Chrome)
1. Abre la URL en Chrome
2. Menú ⋮ → **"Añadir a pantalla de inicio"**

### iOS (Safari)
1. Abre la URL en Safari
2. Botón compartir □↑ → **"Añadir a pantalla de inicio"**

### PC (Chrome / Edge)
1. Abre la URL
2. Icono ⊕ en la barra de direcciones → **"Instalar"**

> 💡 La app funciona **sin conexión** después de la primera carga.

## 🚀 Publicar en GitHub Pages

1. Sube todos los archivos a un repositorio GitHub
2. Ve a **Settings → Pages**
3. En *Source* selecciona `main` / `root`
4. La app estará disponible en `https://TU-USUARIO.github.io/NOMBRE-REPO/`
